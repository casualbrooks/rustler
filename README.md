# Rustler Project

## Prerequisites

1. Install Rust and Cargo:
    ```bash
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    ```

2. Verify installation:
    ```bash
    rustc --version
    cargo --version
    ```

## Running the Project

To run the project:
```bash
cargo run
```
